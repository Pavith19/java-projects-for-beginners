# Student Grade Calculator

A simple Java console application that calculates and displays average grades for a group of students.

## Description

This program allows teachers or administrators to input student data (names and grades), automatically calculates the class average, and generates a comprehensive grade report. It's designed to be straightforward and efficient for quick grade calculations.

## Features

- ✅ Input multiple student names and grades
- ✅ Automatic average grade calculation
- ✅ Clean, formatted grade report display
- ✅ Shows individual student grades alongside class average
- ✅ Simple console-based interface

## Requirements

- Java Development Kit (JDK) 8 or higher
- Any Java IDE (Eclipse, IntelliJ IDEA, NetBeans) or command line

## Installation

1. Clone this repository:
```bash
git clone https://github.com/yourusername/student-grade-calculator.git
```

2. Navigate to the project directory:
```bash
cd student-grade-calculator
```

3. Compile the Java file:
```bash
javac StudentGradeCalculator.java
```

## Usage

1. Run the program:
```bash
java StudentGradeCalculator
```

2. Follow the prompts:
   - Enter the number of students
   - For each student, input their name
   - Enter the corresponding grade (0-100)

3. View the generated report showing:
   - Each student's name and grade
   - Overall class average

### Example

```
Enter the number of students: 3

Enter student 1 name: Alice
Enter Alice's grade: 85

Enter student 2 name: Bob
Enter Bob's grade: 92

Enter student 3 name: Charlie
Enter Charlie's grade: 78

----- Grade Report -----
Alice: 85.0
Bob: 92.0
Charlie: 78.0
------------------------
Average Grade: 85.0
```
